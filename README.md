# laravel-preconfig

LaraFast is a very opinionated Laravel setup script that will create a laravel project.

Configs:

- Git
- Breeze (Livewire)
- Pest

Packages:
- Telescope
- Pulse
- Laradumps
- Laravel Debugbar
- Livewire Toaster

## Requirements

- PHP
- Composer

## Install

```bash
bun add -g lara-fast
```

## Usage

```bash
lara-fast <project-name>
```
